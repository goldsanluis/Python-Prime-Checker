# Prime Number Checker

A Python function that determines whether a given integer is prime,
using an efficient square root-based loop.

## What It Does
- Returns True if the number is prime, False otherwise
- Correctly handles edge cases: 0, 1, and negative numbers return False
- Raises a TypeError if input is not an integer

## Sample Output
is_prime(2)     # True
is_prime(17)    # True
is_prime(21)    # False
is_prime(1)     # False

## Requirements
- Python 3.x
